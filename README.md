Beatbox
=======

Collaborative, realtime, fully reactive drum machine built on top of the [Meteor](https://www.meteor.com/) web framework.

#### Run

Make sure you have Meteor installed

    $ git clone https://github.com/yoavz/beatbox.git
    $ cd beatbox
    $ meteor
